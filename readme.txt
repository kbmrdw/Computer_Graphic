           - Cách 2: 
	+ B1: Mở Visual Studio Code => Chọn File => Chọn Open Folder => Chọn folder DEMO => Chọn folder muốn chạy (Nếu muốn chạy Moving_Boxes thì chọn folder "Moving_Boxes", tương tự cho 2 folder còn lại) 
	+ B2: Mở file index.html => Click chuột phải vào màn hình => Chọn "Open with Live Server" sẽ hiện ra mô hình mô phỏng hình học 3D trên web.
